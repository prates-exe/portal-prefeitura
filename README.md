# 🏛️ Portal Institucional - Prefeitura

Um projeto front-end de um site institucional governamental, desenvolvido com foco em acessibilidade (A11y), UI/UX moderna e componentização visual para portfólio.

## 🔗 Link do Projeto Ao Vivo
[Acessar o Portal da Prefeitura](https://prates-exe.github.io/portal-prefeitura/)

## 💻 Tecnologias Utilizadas
* **HTML5 Semântico:** Estruturação lógica pensando em leitores de tela e boas práticas.
* **CSS3 Avançado:** Uso de CSS Grid, Flexbox e variáveis (`:root`) para criar um Design System escalável e totalmente responsivo (Mobile First).
* **JavaScript (Vanilla):** Lógica de interface, manipulação do DOM para menus responsivos, carrosséis de mídia dinâmicos e filtros de busca em tempo real.

## 🚀 Funcionalidades em Destaque
* **Design Responsivo:** Adaptação fluida e perfeita entre Desktop, Tablet e Mobile.
* **Portal da Transparência (SPA):** Uma página interna que filtra dados via JavaScript instantaneamente, sem recarregar a tela.
* **UI/UX Moderna:** Implementação de "Floating Cards" (cartões flutuantes) na área de busca e tabelas de dados complexos que se transformam em *Cards* no celular para garantir a legibilidade.
